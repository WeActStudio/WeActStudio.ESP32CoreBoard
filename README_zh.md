## 如何下载程序
方法
1. 解压`Tools`目录下的`esptool-v4.5.1-win64_WeActStudio.7z`
2. 运行`WeAct Studio UART Download Tool.bat`
3. 按住BOOT键然后用USB数据线连接电脑，释放BOOT键
4. 拖入需要烧录的程序，输入烧录地址以及串口号，点击回车开始烧录
